# Combinatorial Algorithms
Permutations, Combinations and Variations generator

## Examples

Set = [a, b, c];

## Permutations without repetition
[a, b, c], [a, c, b], [b, a, c],
 
[b, c, a], [c, a, b], [c, b, a]

## Permutations with repetition
[a, a, a], [a, a, b], [a, a, c], 

[a, b, a], [a, b, b], [a, b, c],

[a, c, a], [a, c, b], [a, c, c]

[b, a, a], [b, a, b], [b, a, c],

[b, b, a], [b, b, b], [b, b, c],

[b, c, a], [b, c, b], [b, c, c],

[c, a, a], [c, a, b], [c, a, c],

[c, b, a], [c, b, b], [c, b, c],

[c, c, a], [c, c, b], [c, c, c]

## Combinations without repetition (k = 2)
[a, b], [a, c], [b, c]

## Combinations with repetition (K = 2)
[a, a], [a, b], [a, c], 

[b, b], [b, c], [c, c]

## Variations without repetition (K = 2)
[a, b], [a, c], [b, a], 

[b, c], [c, a], [c, b]

## Variations with repetition (K = 2)
[a, a], [a, b], [a, c], 

[b, a], [b, b], [b, c],

[c, a], [c, b], [c, c]

For more examples take a look at the example.php file.
